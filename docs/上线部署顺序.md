# 小白上线顺序

1. GitHub 上传本目录全部文件。
2. Render 创建 PostgreSQL 数据库。
3. Render 创建 Web Service，Root Directory 填 `backend`。
4. Build Command：`npm install && npm run prisma:generate && npm run build`
5. Start Command：`npm run start`
6. 环境变量至少配置：
   - NODE_ENV=production
   - CORS_ORIGIN=https://citeox.com
   - DATABASE_URL=Render 数据库连接串
   - JWT_SECRET=强随机字符串
   - SESSION_SECRET=强随机字符串
   - ADMIN_EMAIL=你的管理员邮箱或手机号
   - ADMIN_INITIAL_PASSWORD=你的管理员初始密码
7. 部署成功后，把 Render 后端地址发给 Codex，用来接入前端 API。
